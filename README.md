# PIC18F458-Access-Control-Door-Lock-System
An assembly language program for the PIC18F458 microcontroller that implements a secure, keypad-based digital access control system.A secure door lock system built with PIC18F458 assembly, featuring matrix keypad input, LCD feedback, and timer interrupts.
